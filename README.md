**Project Overview**

Save n Serve is a mobile application built with Flutter and Dart that connects food donors with recipients, aiming to minimize global hunger and food waste. It provides a platform for NGOs, hotels, and individuals to donate surplus food to those in need.  ‚û°Ô∏è
